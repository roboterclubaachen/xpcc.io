Member Enumeration Documentation

enum xpcc::lpc::AdcManualSingle::StartCondition
strong

Only in software controlled mode a start condition can be given. If the conversion is started manually with START_NOW only one conversion is made. If a hardware source as a start condition is selected repeated samples can be generated if this condition appears regularly (e.g. repeated timer match)

Enumerator
START_NOW 

Manually start a conversion now.

START_PIO0_2 

Start conversion whenever the selected edge on PIO0_2 occurs.

START_PIO1_5 

Start conversion whenever the selected edge on PIO1_5 occurs.

START_CT32B0_MAT0 

Start conversion whenever the selected edge on CT32B0_MAT0 occurs (timer match)
