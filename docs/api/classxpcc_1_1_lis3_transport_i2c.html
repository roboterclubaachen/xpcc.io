Detailed Description

template<class I2cMaster>
class xpcc::Lis3TransportI2c< I2cMaster >

This class manages communication with the accelerometer via the I2C bus.

To enable I2C mode in the LIS3xx devices, the CS pin must be tied high. However, it is internally pulled-up, so just leaving the CS pin disconnected should work too.

The I2C interface is compliant with Fast Mode (up to 400kHz).
