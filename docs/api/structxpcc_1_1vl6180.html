enum xpcc::vl6180::Start : uint8_t
strong
Enumerator
ModeSelect 

Device Mode select.

0: Ranging Mode Single-Shot, 1: Ranging Mode Continuous

StartStop 

StartStop trigger based on current mode and system configuration of device_ready.

FW clears register automatically. Setting this bit to 1 in single-shot mode starts a single measurement. Setting this bit to 1 in continuous mode will either start continuous operation (if stopped) or halt continuous operation (if started). This bit is auto-cleared in both modes of operation.

enum xpcc::vl6180::VhvRecalibrate : uint8_t
strong
Enumerator
Status 

FW controlled status bit showing when FW has completed auto-vhv process.

0: FW has finished autoVHV operation, 1: During autoVHV operation

Recalibrate 

User-Controlled enable bit to force FW to carry out recalibration of the VHV setting for sensor array.

FW clears bit after operation carried out. 0: Disabled, 1: Manual trigger for VHV recalibration. Can only be called when ALS and ranging are in STOP mode
