Detailed Description

template<typename I2cMaster>
class xpcc::Bma180< I2cMaster >

The BMA180 is a 3-axis accelerometer with high resolution (14-bit) measurement at up to +-16 g. Digital output data is formatted as 16-bit twos complement and is accessible through I2C digital interface.

The following LSB-values are valid: 1 LSB{ADC} = 0.25mg in 2g range; it scales with range 1 LSB{TEMP} = 0.5 degrees Celsius

For further information on the special sensing functions, consult the datasheet.
