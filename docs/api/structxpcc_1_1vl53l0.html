enum xpcc::vl53l0::Start : uint8_t
strong
Enumerator
StartStop 

StartStop trigger based on current mode and system configuration of device_ready.

FW clears register automatically. Setting this bit to 1 in single-shot mode starts a single measurement. Setting this bit to 1 in continuous mode will either start continuous operation (if stopped) or halt continuous operation (if started). This bit is auto-cleared in both modes of operation.
