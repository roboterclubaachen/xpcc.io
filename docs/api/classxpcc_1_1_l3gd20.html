Detailed Description

template<class Transport>
class xpcc::L3gd20< Transport >

The L3GD20 is a low-power three-axis angular rate sensor. The L3GD20 has a full scale of ±250/±500/±2000 dps and is capable of measuring rates with a user-selectable bandwidth.

Template Parameters
TransportEither the I2C or SPI Transport Layer.
