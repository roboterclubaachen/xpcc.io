Detailed Description

Most of the time, 256 (8bit) steps are enough for simple indication, however, for color sensitive tasks (like RGB LEDs) consider using the 1000 step table. Only the tables you use will be copied into flash.

If any special behavior is expected, you can copy and modify these tables into your own project. All tables use a gamma of 2.2.
