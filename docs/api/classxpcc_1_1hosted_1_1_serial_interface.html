Detailed Description

The normal life cycle should look like as follows:

  • Construct the interface via SerialInterface()
  • setDeviceName()
  • setBaudRate()
  • open()
  • Read & write, whatever you want... Note: Use bytesAvailable() before read operation.
  • close()
